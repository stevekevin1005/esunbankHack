使用流程:

初步規劃:

用戶A B 輸入自己銀行帳戶密碼 登入驗證好後存入手機內部

A 輸入應付金額後產生二維條碼
B 掃條碼
A 通知 用戶XXXX 準備向您收取XXX費用
A 按確定
A-->轉帳-->B
